    1. What is the difference between an inline element and a block element? 
    A block-level element always starts on a new line and takes up all available space on page.
   An inline element does not start on a new line and only takes up as much width as required.

    5. In your own words, explain the box model. What is the fix for the box model? 
    It is a pattern for positioning HTML elements on the page using CSS. We can consider all html elements to be boxes. Each box consist of: content, padding, border, margin- make up the box model
